# ✨ 优势与不足

## 一、优势

### 1. 潜力无穷

由于基于 CoCo 源代码计划开发，理论上可以修改任意 CoCo 代码。~~实际上根本看不懂 CoCo 的代码~~

## 二、不足

### 1. 开发周期长

CoCo Next 开发难度极大：

- CoCo 源代码计划尚处在早期阶段，很多代码需要先在 CoCo 源代码计划还原，这需要开发者熟悉逆向工程。
- CoCo 代码多且杂，还使用了 React、Redux 等多种复杂框架。

CoCo Next 由社区用户维护，社区用户大多是学生，能够用于开发的时间极少。

### 2. 稳定性差

由于经常更新、代码杂乱、测试不方便等多重原因，CoCo Next 稳定性极差，经常出现一些严重问题，如：打不开作品。
