# ⇆ 与其他项目的区别

优化 CoCo 编辑器的项目有很多，由于使用的技术不同，CoCo Next 与这些项目存在较大的区别。

## 1. 与 CoCo Pro 的区别

CoCo Next 基于 CoCo 源代码计划构建，通过直接修改编辑器代码的方式修改 CoCo，CoCo Pro 通过修改全局 API 和 CoCo 编辑器的分发文件的方式修改 CoCo。

相较于 CoCo Pro，CoCo Next 先使用逆向工程技术还原编辑器代码，这使得代码更易于查看和编辑。

## 2. 与 Better CoCo 的区别

Better CoCo 是一个集成了多个优化的浏览器扩展程序，它并没有修改 CoCo 编辑器的代码，所以可以直接使用 CoCo 的成果。

也正是因为没有修改 CoCo 编辑器的代码，Better CoCo 不太可能深度修改 CoCo。

CoCo Next 相对独立于 CoCo，不能使用一些 CoCo 的资源（如：编程猫的 CDN），但具有深度修改 CoCo 的潜力。
