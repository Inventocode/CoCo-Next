# 📕 简介

CoCo Next 是基于 [CoCo 源代码计划](https://gitee.com/oldsquaw-coco-next/CoCo-Source-Code-Plan) 二次开发的 CoCo，旨在通过直接修改编辑器代码的方式修复 CoCo 的缺陷、增强 CoCo 的功能。

CoCo 是编程猫自研的国内首款面向教育领域的 图形化手机应用编辑器 ，低门槛、跨平台的特性使得不懂编程的人也可以制作出专业的APP，从而解决现实问题、实现创意。

由于 CoCo 功能不完善且长期不更新，CoCo 用户 SLIGHTNING 创建了 CoCo 源代码计划 和 CoCo Next，以继续更新维护 CoCo，详见 [产生背景](./background)。
