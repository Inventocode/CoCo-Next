# 📏 发展规划

## 第零阶段：保障 CoCo 功能

保证 CoCo 功能能够正常使用，使用户可以从 CoCo 无缝迁移。

## 第一阶段：补全基础功能

在该阶段，我们会致力于 CoCo 的一些基础问题，如：Undefined 修复、移除自定义控件。这些功能虽然大多已有其他解决方案，但是用起来太麻烦了。同时，这也是社区工具的基础功能。

## 第二阶段：提升可扩展性

可扩展性是提升编辑器用途广泛程度、社区活跃度的关键。我们计划通过自定义控件和插件系统来提升可扩展性。

## 第三阶段：实现高级功能

这个阶段纯是大饼，一点也实现不了。

实现高级功能，如：布局功能、预览热更新。
