# 🐱 登录编程猫账号

CoCo Next 部分功能需要登录编程猫账号才能使用。

:::info
关于 CoCo Next 如何代理编程猫服务，请参考 [技术分析](../../tech/fork/proxy)。
:::

## 一、当前域登录

你可以在主页或编辑器页面点击右上角的 登录 按钮来登录编程猫账号。

:::tip
登录界面可能加载较为缓慢，点击按钮后，如果没有反应或提示登录界面弹出失败，可以多试几次。
:::

## 二、原始域登录

一些功能（如：协作、云房间、某改云控件）可能需要原始域登录才能使用。

:::tip

以下情况不需要进行原始域登录：

- 不使用依赖 WebSocket 的服务，如：协作、云房间、某改云控件；
- WebSocket 可以被代理的页面，如：使用本地服务（coco-next.localhost）、本地 APP 时。

:::

在当前域登录之后，在主页或编辑器页面，点击右上角用户头像，在弹出菜单上点击原始登录按钮，即可弹出原始域登录界面。

要退出原始域登录，请在原始账号设置页面（可以从主页右上角用户头像菜单中打开）退出登录。
