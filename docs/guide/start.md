# ⚡️ 快速开始

## 一、本地使用

:::warning
本地使用只是把 CoCo Next 的部分资源存储到了本地，仍有部分资源需要从网上加载。
:::

目前仅支持 Windows 系统。

到 [Release](https://gitee.com/oldsquaw-coco-next/CoCo-Next/releases) 界面下载 CoCo Next 服务，下载完成后双击程序启动。

## 二、在线使用镜像仓库

在线使用可能存在加载缓慢、无法登录等问题。

镜像仓库：https://github.com/Inventocode/CoCo-Next

- Cloudflare Pages：https://next.ccwidget.top/
- Github Pages（较慢）：https://coco.ccwidget.top/
