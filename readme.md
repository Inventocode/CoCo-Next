# CoCo Next

下一代 CoCo 编辑器。

## 一、介绍

基于 [CoCo 源代码计划](https://gitee.com/oldsquaw-coco-next/CoCo-Source-Code-Plan) 二次开发的 CoCo，旨在通过直接修改编辑器代码的方式修复 CoCo 的缺陷、增强 CoCo 的功能。

具体介绍详见 [关于我们 | CoCo Next 文档](https://next.ccwidget.top/docs/about/intro)。
